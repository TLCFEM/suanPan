Please visit https://tlcfem.github.io/suanPan-manual/latest/ for user manual and examples.

Please ensure your CPU supports AVX2. The binary is compiled with AVX2. It can only be executed on platforms that support AVX2.

Please use a modern terminal such as Windows Terminal and Fluent Terminal for better output display.

Please find and execute AddAssociation.bat with admin privilege to add Sublime Text auto-completion and syntax support. Alternatively, you may want to add the installation folder to PATH variable.
